## ADDED Requirements
### Requirement: 创建页标题区设计
创建页面 SHALL 在顶部显示标题区，包含主标题和副标题，居中排列，下方留白充足，视觉轻盈透气。

#### Scenario: 标题区展示
- **WHEN** 用户访问 `/create` 页面
- **THEN** 页面顶部显示主标题"记一次告别"（字号最大，字重细，深灰紫色）和副标题"Record a Goodbye"（字号较小，字重偏细，浅灰紫色），居中排列，下方留白充足

### Requirement: 过渡视觉元素
创建页面 SHALL 在标题区下方显示一个低透明度的灰色菱形图标，作为视觉缓冲，让标题与输入区的衔接更自然。

#### Scenario: 菱形图标显示
- **WHEN** 用户访问 `/create` 页面
- **THEN** 标题区下方显示一个低透明度的灰色菱形图标，居中显示，作为视觉缓冲

### Requirement: 引导文字
创建页面 SHALL 在输入框上方显示引导文字"你想告别什么？"，引导用户进入输入环节。

#### Scenario: 引导文字显示
- **WHEN** 用户访问 `/create` 页面
- **THEN** 在输入框上方显示引导文字"你想告别什么？"，衬线字号适中，居中，字重常规，浅灰紫色

### Requirement: 输入框样式
创建页面的输入框 SHALL 占据横向全屏宽度，圆角设计柔和，占位符为"关系/情绪/旧物/过去的自己……"。

#### Scenario: 输入框样式和占位符
- **WHEN** 用户访问 `/create` 页面
- **THEN** 输入框占据横向全屏宽度，圆角设计柔和，占位符显示为"关系/情绪/旧物/过去的自己……"（浅灰色衬线文字）
