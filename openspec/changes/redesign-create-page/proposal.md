# Change: 重新设计创建页标题区和视觉层次

## Why
当前创建页的标题和视觉层次需要优化，使其更加轻盈透气，视觉上更符合应用的克制、温柔风格。标题区需要更清晰的层次结构，并通过视觉元素（菱形图标）增强过渡效果，提升用户体验。

## What Changes
- 标题区文案和样式调整：
  - 主标题从"告别博物馆"改为"记一次告别"，字号最大，字重细，深灰紫色
  - 副标题从"创建一个"白膜"。用可控的积累，完成一次体面的告别。"改为"Record a Goodbye"，字号较小，字重偏细，浅灰紫色
  - 标题区下方留白充足，居中排列，视觉轻盈透气
- 添加过渡视觉元素：
  - 标题下方放置一个低透明度的灰色菱形图标，作为视觉缓冲，让标题与输入区的衔接更自然
- 引导文字优化：
  - 在输入框上方添加引导文字："你想告别什么？"，衬线字号适中，居中，字重常规，浅灰紫色
- 输入框样式调整：
  - 输入框占据横向全屏宽度，圆角设计柔和
  - 输入框占位符改为"关系/情绪/旧物/过去的自己……"，使用浅灰色衬线文字

## Impact
- 受影响的能力：create-page（如果存在规范）或 mvp 规范中的创建页部分
- 受影响的代码：
  - `src/pages/CreatePage.tsx` - 更新标题区、添加菱形图标、调整引导文字和输入框样式
